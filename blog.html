Let's explore some important topics about javascript!

sessionStorage is mostly similar to localStorage; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.

The main difference between Global and local scope is that global scope can be accessed globally in the entire program, whereas block scope restricts the variable that is declared inside a specific block, from access by the outside of the block.
